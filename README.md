## 🚀 How to Run the Solutions Locally

If you want to run or test these solutions on your machine, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/mkb-2007/LeetCode-Problems.git](https://github.com/mkb-2007/LeetCode-Problems.git)
    cd LeetCode-Problems
    ```

2.  **Compile the Java file:**
    ```bash
    javac "Day 1 - 7 - Reverse the Integer.java"
    ```

3.  **Run the program:**
    *(Note: To run it locally, you may need to add a `main` method to pass sample inputs to the solution class.)*
    ```bash
    java Solution
    ```
